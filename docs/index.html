Loadout

Windows Infrastructure Workstation Setup Tool — installs, updates, and manages a curated set of tools for infrastructure professionals.

Install

Open an elevated PowerShell prompt and run:

irm https://bferguson-dev.github.io/Loadout/install | iex

Loadout installs to %APPDATA%\Loadout and self-updates on every launch.
